                        function generateNumbers(svalue, evalue){
                            let numbers=[];
                            //get all numbers from start to end 
                            for(let i=svalue; i<=evalue;i++){
                                numbers.push(i);
                            }
                        
                            return numbers;
                        }

                        function displayNumbers(numbers){

                                let element=numbers[i];

The Code is structed in three distinct functions

Get Values

Get values is an start function that is called from the UI to collect the start value and end value inputs from the html form and converts them to integers in order to be operated, it also calls the functions generateNumbers() and displayNumbers() to complete the program

Generate Numbers

Input:
integer- Start Value,integer -End Value
Function:
Generate numbers is a loop with start and end values given by the user. In addition to generating the numbers, the program will store them into an array called numbers
Output:
integer array -numbers

Display Numbers

Input:
integer array -numbers Function:
Loop over the number array and identify the even and odd values using the modulo (%) operator. The function also generates the html code required to complete the table number in the UI
Output:
None
